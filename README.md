# Digital Clock - Multi-Timezone Display

A web-based digital clock application that displays the current time in multiple time zones simultaneously.

## Features

- 🕐 Real-time digital clock display
- 🌍 Multiple timezone support
- 🎨 Clean, modern UI
- ⚡ Live updates every second
- 📱 Responsive design

## File Structure

```
├── index.html          # Main HTML file
├── style.css           # Styling
├── script.js           # Clock logic and timezone handling
└── README.md           # This file
```

## Supported Time Zones

The clock displays time for:
- UTC (Coordinated Universal Time)
- EST (Eastern Standard Time)
- CST (Central Standard Time)
- MST (Mountain Standard Time)
- PST (Pacific Standard Time)
- IST (Indian Standard Time)
- JST (Japan Standard Time)
- GMT (Greenwich Mean Time)

## How to Use

1. Open `index.html` in your web browser
2. The clock will automatically display the current time in all configured time zones
3. Times update in real-time every second

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (no dependencies)
